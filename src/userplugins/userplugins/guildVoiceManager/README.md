# GuildVoiceManager

Mute/unmute local par rôle pour les events de guilde dans les canaux vocaux Discord.

## Commandes

| Commande | Action |
|----------|--------|
| `/atk` | Mute DEF + ROM — tu n'entends que les ATK |
| `/def` | Mute ATK + ROM — tu n'entends que les DEF |
| `/rom` | Mute ATK + DEF — tu n'entends que les ROM |
| `/unmute` | Unmute tout le monde |
| `/mute?` | Liste les users mutés |
| `/vdebug` | menu de maintenance |

## Configuration

Les noms des rôles ATK, DEF et ROM sont configurables dans les paramètres du plugin.
Ils doivent correspondre exactement aux noms de role asssignés dans Discord.
